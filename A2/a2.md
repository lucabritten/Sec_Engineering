## Hashwerte von /etc/services berechnen

```bash
    sha256sum /etc/services
```

e4ab19dc7199f9f15fac3d9cb62743637950cfab70dd9238715ad9d73a654042

```bash
    sha1sum /etc/services
```

1e08304020445c121b7ae2c6810d5dbe9590c835

```bash
    sha384sum /etc/services
```

876e207bb22086b7b4e2fe030274034a5578c7225c6b259041de83c61c7860c3d60d4053e20e63aede1e58da36d6a14f

```bash
    md5sum /etc/services
```

b373009fdd7e9a9a2b266c2044fb7948

Die Algorithmen unterscheiden sich in ihrer Hashlänge: MD5 (128 Bit), SHA-1(160 Bit). Diese Algorithmen gelten aufgrund **höherer Kollisionswahrscheinlichkeit** als unsicher. Standard sind aktuell Algorithmen wie sha256. Größere Hashlängen erhöhen die Sicherheit, können aber je nach Alogrithmus auch den Rechenaufwand erhöhen.
